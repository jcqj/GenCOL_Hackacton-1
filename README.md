# Nexus

###### Artículos Deportivos

#### Hackathon 1 - SALA 9



##### Descripción del proyecto

Este proyecto es una Tienda Online Interactiva desarrollada durante la Hackathon 1 de Generation Colombia. Nuestra misión fue construir una landing page funcional donde los usuarios puedan explorar productos, gestionarlos en un carrito de compras y contar con persistencia de datos.



La categoría elegida es: **Deportes**



##### Factor de Innovación

Para este apartado decidimos implementar un producto como Propuesta de Valor, las propuestas de desarrollo suelen implementar el Stack estándar de la industria, incluyendo el producto de Ropa Deportiva Personalizada atacamos a un mercado amplio pero con poca oferta en la actualidad.



##### Requisitos Técnicos Implementados

* HTML5 Semántico: Estructura jerarquizada para mejor accesibilidad y SEO.
* Bootstrap \& CSS: Diseño 100% responsivo adaptable a móviles, tablets y escritorio.
* JavaScript: Renderizado dinámico de productos y lógica modular del carrito.
* LocalStorage: El carrito persiste incluso al cerrar o recargar la pestaña.
* Animaciones: Transiciones aplicadas para mejorar la experiencia de usuario.
